Notes
-----

.. note::
   - Tested against SmartCS NS-2250 System Software Ver 2.1



Examples
--------

.. code-block:: yaml

    - name: Login to SmartCS and execute "show version"
      seiko.smartcs.smartcs_tty_command:
        tty: 1
        cmd_timeout : 5
        recvchar:
        - 'login: '
        - 'Password: '
        - 'SWITCH> '
        sendchar:
        - __NL__
        - user01
        - secret01
        - show version
